# Next.js AI-First Starter Kit

Production-ready starter for business applications (SaaS, CRM, internal systems).

## Tech Stack

- **Framework:** Next.js 16 + App Router + Turbopack
- **Language:** TypeScript (strict)
- **Styling:** Tailwind CSS v4 + shadcn/ui
- **Database:** SQLite (better-sqlite3) via Drizzle ORM
- **Auth:** Better Auth
- **Forms:** React Hook Form + Zod
- **i18n:** next-intl (ru / en)
- **Testing:** Vitest + Playwright

## Quick Start

```bash
npm install
cp .env.example .env
npm run dev
```
